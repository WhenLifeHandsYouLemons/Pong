HOW TO SETUP GAME:
1. Create a folder in "Local Disk (C:)" called "Sooraj Pong Game" (without the quotation marks but with the spaces).
2. Add ALL the files included into that folder you just created.
3. Then run the file called "Pong.py" (without the quotation marks) inside of the folder to play the game.
